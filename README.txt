
Hướng dẫn cài đặt và build APK (Gia đình BinBit)

1. Cài Android Studio tại: https://developer.android.com/studio
2. Mở Android Studio > Open Project > chọn thư mục này
3. Copy lại file google-services.json của bạn vào thư mục /app
4. Đảm bảo đã đăng nhập tài khoản Google có Maps + Firebase
5. Nhấn Build > Build APK (hoặc Build Bundle)
6. APK sẽ xuất hiện trong thư mục /app/build/outputs/apk/

Lưu ý:
- Mã PIN: 3112 để đổi cấu hình
- Tên app: Gia đình BinBit
- Gửi vị trí mỗi phút, lưu Firebase
